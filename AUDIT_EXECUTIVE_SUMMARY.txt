🔴 CRITICAL ISSUE #1: ActionLog.CreatedBy

Problem:
  Entity property CreatedBy is defined as Guid
  Database column is NVARCHAR(255) (string)
  All stored procedures expect NVARCHAR(255) string parameter
  Type mismatch causes InvalidCastException at runtime

Impact:
  ActionLog.Add() operations will fail
  ActionLog.Update() operations will fail
  System cannot log actions correctly
  BLOCKING ISSUE

Solution:
  Change WebAPI/Entities/ActionLog.cs line 9
  FROM: public Guid CreatedBy { get; set; }
  TO:   public string CreatedBy { get; set; }

Verification:
  Build: dotnet build (should succeed)
  Test: ActionLog CRUD operations (should all work)
  Deploy: Single-line change, low risk
